Guide d’utilisation
===================

Cas d’usage principaux
----------------------

L’utilisateur peut :

- consulter la page d’accueil
- afficher la liste des locations
- consulter le détail d’une location
- afficher la liste des profils
- consulter le détail d’un profil

L’application propose également des pages d’erreur personnalisées
pour améliorer l’expérience utilisateur en cas de page introuvable
ou d’erreur interne.
